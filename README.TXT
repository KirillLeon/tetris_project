TETRIS
Тетрис - это однопользовательская игра-головоломка. Целью игры является заполнение 
максимального возможного количества линий. Игрок как и в оригинальной версии игры 
может двигать фигурку-тетрамино влево и вправо, также он может поворачивать её на 90 
градусов по часовой стрелке и ускорять с помощью специальных кнопок. 
При заполнении сразу нескольких линий игрок получает дополнительные очки. 
Максимальное количество набранных очков является рекордом и записывается в 
отдельный файл, так что при повторном запуске игры все данные будут сохранены. 
В игре также есть управление звуком.

Регламент начисления очков:
1 линия: 100
2 линии: 300
3 линии: 700
4 линии: 1500

Кнопки управления:
Стрелки вправо, влево для управления
фигурой по горизонтали.
Стрелка вверх для поворота.
Стрелка вниз для ускорения.
Кнопка 1 включение/выключение музыки 
Кнопка 2 уменьшение громкости в 2 раза
Кнопка 3 увеличение громкости в 2 раза
